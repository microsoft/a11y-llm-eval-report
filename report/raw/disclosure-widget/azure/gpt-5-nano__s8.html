What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the web. It defines the elements that compose a web page, such as headings, paragraphs, lists, links, images, and more.

Why use the details/summary pattern?

The details/summary pattern provides a simple, accessible way to show and hide content without writing extra JavaScript. It works across modern browsers and is easy to style.
