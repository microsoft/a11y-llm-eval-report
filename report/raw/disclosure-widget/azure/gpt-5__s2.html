Native Details/Summary

What is the details/summary element?

The HTML <details> element creates a disclosure widget—a simple, accessible expand/collapse. The <summary> provides the visible trigger, and the rest of the content becomes visible when opened.

  • Built-in keyboard and screen reader support
  • No JavaScript required
  • Easily styled with CSS
