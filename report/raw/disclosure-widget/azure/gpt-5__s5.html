Expand/Collapse Widgets

Native details/summary widget

What is an expand/collapse widget?

An expand/collapse widget allows users to reveal additional content on demand. The native HTML details element provides this behavior with built-in accessibility and keyboard support.

<details class="details"> <summary>Click to toggle</summary> <p>Hidden content.</p> </details>
Tip: The details element can be pre-opened by adding the open attribute.

Custom toggle button controlling a region

This collapsible region is controlled by a standard button. When collapsed, it uses the hidden attribute to remove the content from the accessibility tree and tab order.

  • Accessible via aria-expanded and aria-controls
  • Keyboard-friendly: Space/Enter toggles, focus remains on the button
  • Programmable: You can expand/collapse from JavaScript
// Toggle via JavaScript: document.getElementById('toggle-1').click();
