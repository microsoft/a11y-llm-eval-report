Expand/Collapse Widgets

Examples of native and custom expand/collapse patterns. Each widget is wrapped with an "example" container and the controlled content uses a "details" class.

Native Details Element

What is an expand/collapse widget?

An expand/collapse widget lets users reveal additional content when they choose. The native HTML details/summary element provides a built-in accessible option with keyboard and screen reader support.

  • Click the summary to toggle visibility.
  • Press Space or Enter when the summary is focused.
  • State is indicated via the element's open attribute.

Custom Toggle Button (ARIA Controlled)

Toggle to reveal extra information

This custom widget uses a button with aria-expanded and aria-controls to manage visibility of the controlled region below.

Because it is custom, JavaScript updates the button state and toggles the hidden content. The content container is a div with class "details".
