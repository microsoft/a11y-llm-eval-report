Expand / Collapse Widget Demo

Example 1: Simple Text

This panel reveals additional information when expanded. It uses a simple button to toggle visibility and is accessible with ARIA attributes.

  • Accessible using: aria-expanded on the trigger
  • Content is wrapped in a container with class "details"
  • Multiple examples can be included on one page

Example 2: Collapsible Section with Image

This example includes an image and a short paragraph. The content area has a class of "details" as requested.

Images and other rich content can be placed inside the details container.
