Widget Example One

Click to learn about HTML

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications.

With HTML, you can create and structure sections, paragraphs, headings, links, and all kinds of content for a web page.

  • HTML elements are represented by tags.
  • Browsers do not display the HTML tags, but use them to render the content of the page.

Widget Example Two

Click for CSS basics

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML, or XHTML).

CSS describes how elements should be rendered on screen, on paper, in speech, or on other media. It saves a lot of work. It can control the layout of multiple web pages all at once.

Key concepts:

  1. Selectors: Used to "find" (or select) the HTML elements you want to style.
  2. Properties: The aspects of an element you want to change (e.g., color, font-size).
  3. Values: The specific setting for a property (e.g., red, 16px).

Widget Example Three (Initially Open)

This widget is open by default

You can make a <details> element open by default by adding the open attribute to it, like so: <details open>.

This is useful for displaying important information or a default section that users often need to see.
