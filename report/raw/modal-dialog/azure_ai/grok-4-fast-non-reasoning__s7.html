Sample Modal Dialog

This is an example of a modal dialog component. It includes a header with a title and close button, a body with content, and a footer with action buttons.

Modals are great for:

  • Confirming user actions
  • Displaying important information
  • Collecting user input
  • Presenting forms or settings

The modal is accessible and includes proper ARIA attributes for screen readers.
