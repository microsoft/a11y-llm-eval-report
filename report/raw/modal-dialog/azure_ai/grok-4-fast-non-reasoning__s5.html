Example Modal

This is an example modal dialog. It demonstrates a clean, accessible modal component that can be triggered by a button with the trigger class.

Features:

  • Opens and closes with smooth animations
  • Keyboard accessible (ESC to close, Tab to navigate)
  • Responsive design for mobile devices
  • Proper ARIA attributes for screen readers
  • Overlay backdrop prevents interaction with background content

You can customize the content, styling, and behavior as needed for your application.
