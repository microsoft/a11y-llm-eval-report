Modal dialog example

This example demonstrates an accessible modal dialog. The trigger button has the trigger class and the dialog is closed by default.

There may be page content behind the modal. When the modal opens, focus is moved into it and keyboard navigation is trapped.

Example Modal

This is an example of a modal dialog. It is hidden when the page loads. You can close it by:

  • Clicking the close button
  • Clicking outside the dialog (the overlay)
  • Pressing the Escape key
