Modal Dialog Component (Closed by default)

This example demonstrates a modal dialog that's closed by default. Use the button below to open it. The open button has class "trigger".

Example Modal

This is an accessible modal dialog. It is closed by default and can be opened by the button with the trigger class.

While open:

  • Focus is trapped inside the modal.
  • Press Esc to close.
  • Click outside the dialog (the overlay) to close.
