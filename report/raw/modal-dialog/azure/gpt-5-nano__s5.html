Modal Dialog Component

This page demonstrates a modal dialog component. The modal is closed by default. Click the button below to open it. The button to open the modal has a trigger class.

They can close the modal by: - Clicking the overlay outside the dialog - Clicking the Close button inside the dialog - Pressing the Escape key - Trapping focus within the modal while it is open
