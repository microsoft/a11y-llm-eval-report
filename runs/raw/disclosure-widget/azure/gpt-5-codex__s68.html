Expand/Collapse Widget Examples

Overview

This widget demonstrates a simple expand/collapse interaction controlled by a button. It updates accessibility attributes to indicate the current state to assistive technologies.

The content is hidden by default and becomes visible when the user activates the control.

Checklist
  1. Click the button labeled “Reveal Steps”.
  2. Observe the content area smoothly appear beneath the control.
  3. Click the button again to hide the content.
  4. Notice the button label and icon change to reflect the current state.
