Native details/summary widget

Project Overview

This panel uses the native HTML details/summary pattern. It is accessible by default and supports keyboard and mouse interactions.

Use cases include FAQs, disclosure of advanced options, and progressive detail reveal.

  • Keyboard friendly
  • No JavaScript required
  • Respects user agent defaults

Button-controlled expandable panel

  • Reduced initial load time by ~15%
