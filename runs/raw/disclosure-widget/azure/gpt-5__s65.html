Expand/Collapse Widgets

Examples of interactive panels you can toggle open and closed.

Native details/summary

What is an expand/collapse widget?

An expand/collapse widget reveals additional content on demand, helping reduce visual clutter. The native details element provides this behavior without custom JavaScript.

  • Keyboard and screen reader support is built in.
  • Use the summary element for the toggle control.
  • Place any expanded content after the summary.

Button-controlled panel

This panel is controlled by a button using aria-expanded and aria-controls. Clicking the button toggles the state and reveals content.

  • Declarative: the button declares which region it controls.
  • Stateful: aria-expanded reflects open/closed.
  • Flexible: works for any content container.
