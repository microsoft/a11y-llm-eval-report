Expandable / Collapsible Widgets

Demonstration of an accessible expand/collapse widget. Each widget is wrapped in a container with class "example". The content container uses the class "details".

An expand/collapse widget lets users reveal or hide content on demand. This helps manage long pages by presenting a concise summary and providing the option to view more details when needed.

Accessibility considerations include using a native button for focus and keyboard interactions, and tying the button to the content with aria-expanded and aria-controls attributes.

Click the header button to expand or collapse the content. The state is indicated by the aria-expanded attribute on the button, and the panel is shown or hidden accordingly.

  • Press Enter or Space when the button is focused to toggle (default button behavior).
  • Use the arrow indicator on the right of the header to see the current state.
