Example Modal Dialog

This is an example of a modal dialog component. It features:

  • A clean, responsive design
  • Smooth animations
  • Keyboard accessibility (ESC to close)
  • Overlay backdrop
  • Close button and action buttons

Modals are great for confirmation dialogs, forms, or any content that needs user attention without navigating away from the current page.
