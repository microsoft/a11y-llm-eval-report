Modal Dialog Component

This is an example of a modal dialog. It is closed by default. Use the button below to open it.

Tip: You can close the modal by pressing Esc, clicking on the overlay, or using the close button.

Example Modal

This modal is a generic component that supports opening via a button with the "trigger" class, closing with the close button, clicking outside on the overlay, or pressing the Escape key.

It also manages focus while open and returns focus to the trigger when closed.
