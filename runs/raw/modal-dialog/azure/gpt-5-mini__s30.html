Modal Dialog Component Demo

This page demonstrates an accessible modal dialog. The modal is closed by default. Use the button below to open it.

Example Modal

This is a simple modal dialog example. It demonstrates accessible behaviors:

  • Closed by default.
  • Opens from a trigger button with class "trigger".
  • Closes with the close button, by clicking the overlay, or pressing Escape.
  • Traps focus while open and restores focus on close.
