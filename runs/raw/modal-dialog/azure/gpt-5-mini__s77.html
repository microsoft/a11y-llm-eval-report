Example Modal

This is a sample modal. It demonstrates:

  • Opening from a button with class "trigger"
  • Focus being trapped inside the dialog while open
  • Closing via the close button, overlay click, or Escape key
  • Returning focus to the element that opened the dialog

Try tabbing through the dialog and pressing Escape.
