Expand / Collapse Widgets Demo

ARIA-controlled widget

This widget uses a button to control the visibility of the content. It demonstrates accessible ARIA attributes for expand/collapse functionality.

  • Accessible via aria-expanded and aria-controls
  • Content area uses class="details" as the controlled container
  • Animated height transitions for a smooth UX

CSS-only widget

This widget demonstrates a purely CSS-based expand/collapse interaction using a hidden checkbox to toggle the content.

  1. No JavaScript required for the toggle behavior
  2. Content container uses class="details"
  3. Accessible enough for simple use cases
