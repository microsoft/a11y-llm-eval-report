Expand/Collapse Widget

Each widget is wrapped in a div.example. The controlled content container uses the details class. Toggle with mouse, touch, or keyboard (Enter/Space).

Example 1 — Single panel

A simple expand/collapse pattern using a button that controls a content region. The region is hidden with the hidden attribute and announced via aria-expanded.

  • Button toggles visibility
  • Chevron rotates when open
  • Keyboard accessible
Example 2 — Another panel

This example demonstrates multiple independent widgets on the same page. Each button uses aria-controls to reference its corresponding .details region.

  • Independent state per widget
  • Uses event delegation for simplicity
  • Respects reduced motion preferences (no heavy animations)
Example 3 — With extra content

You can place any HTML inside the controlled region: text, lists, forms, or media. The only requirement here is that the container has the details class.

Tip: If you want an accordion (only one open at a time), you can close siblings when opening a panel.
