Expand / Collapse Widget Examples

Accessible widgets that show and hide content. Each example is wrapped in a div.example and the controlled content has the class "details".

This is additional information revealed when the toggle is expanded. It is wrapped in an element with class="details".

Useful for progressive disclosure of content.

Q: How does this widget work?

A: JavaScript toggles aria-expanded on the button and adjusts the details container's max-height for a smooth animation. The container has class="details".

Follow these steps:

  1. Open the widget using the button.
  2. Read the content. Screen readers will announce the expanded state.
  3. Close when finished.

Keyboard: Tab to the button, Enter or Space to toggle.
