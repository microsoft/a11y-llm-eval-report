Expand / Collapse widget examples

Each widget is wrapped in a div.example. The controlled content container has the class "details". The buttons use aria-expanded and aria-controls for accessibility.

This is the controlled content for the basic example. It starts collapsed. Use the button above to expand and reveal the content. The container has the class "details".

Tip: Buttons use aria-expanded and aria-controls so assistive technologies know the state and relationship.

Built-in features

  • Accessible ARIA attributes (aria-expanded, aria-controls, role="region").
  • Smooth height animation when opening and closing.
  • Works with keyboard (button, enter/space) and screen readers.
