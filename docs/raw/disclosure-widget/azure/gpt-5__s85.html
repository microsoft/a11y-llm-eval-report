Expand/Collapse Widgets

Demonstrating native and custom disclosure patterns. Each widget is wrapped with an example container and uses a details class for controlled content.

Native details element

What is an expand/collapse widget?

An expand/collapse widget lets you reveal additional information on demand. The native details element provides built-in accessibility and keyboard support.

  • Press Enter or Space on the summary to toggle.
  • Works without JavaScript.
  • Can be nested or repeated.

Custom disclosure with button

This panel is controlled by a button using aria-expanded and aria-controls. The container for controlled content has a details class.

  • Accessible: the button indicates its expanded state.
  • Flexible: can include any markup.
  • Stylable: full control over transitions and layout.
