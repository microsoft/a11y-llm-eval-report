Smooth animation

Animated content
This example animates height for a smooth transition.

Animations are performed by measuring the content's full height and transitioning the max-height CSS property. The script ensures the height is set to "none" after expanding so the content can grow naturally.

If the user prefers reduced motion, the transition is disabled via the prefers-reduced-motion media query.

This demonstrates a common pattern for accessible expand/collapse widgets: use a button with aria-expanded and aria-controls, and set role="region" and aria-labelledby on the controlled content.
