This is a basic example of an expand/collapse widget. The button controls the visibility of this content via aria-expanded and aria-hidden.

Try clicking the header or using Space/Enter while it's focused.

This widget starts expanded. The script will calculate the content height and allow a smooth transition when collapsing, and it will remove the inline max-height after expansion so content can grow naturally.

  • Accessible: aria-expanded on the button
  • Accessible: aria-hidden on the content region
  • Smooth animation with respect to user's reduced-motion preference

This example shows an expander that contains another expand/collapse widget inside it.

Inner content can be toggled independently. The outer container will animate its height to accommodate the inner content when expanded.
