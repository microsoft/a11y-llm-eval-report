Tutorial: Creating an Expandable Section

  1. Create a container element that will hold the trigger and the hidden content.
  2. Add a button or summary element with aria-expanded and aria-controls attributes.
  3. Place the content to reveal inside a div with the details class and set the hidden attribute.
  4. Use JavaScript to toggle the hidden attribute and update aria-expanded accordingly.

Frequently Asked Questions

How do I integrate the widget?

Import the script and styles into your project, then add the HTML structure shown above. The widget automatically discovers toggle buttons within any example container.

Can I have multiple widgets on a page?

Yes. Each widget manages its own state independently, so you can collapse one while another remains open.
