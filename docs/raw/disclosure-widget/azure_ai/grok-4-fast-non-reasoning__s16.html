Q: How does the expand/collapse work?

A: The widget uses JavaScript to toggle a CSS class that controls the max-height property, creating a smooth expand/collapse effect.

Q: Is it accessible?

A: Yes! The widget includes proper ARIA attributes, keyboard navigation, and works with screen readers.

Q: Can I customize the animation?

A: Absolutely. You can modify the CSS transition properties and timing functions to match your design requirements.
