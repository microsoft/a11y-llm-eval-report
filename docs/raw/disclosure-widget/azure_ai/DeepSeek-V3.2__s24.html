Example 3: FAQ Style

This widget uses JavaScript to toggle a CSS class that controls the visibility of the content area. The transition effect is achieved with CSS transitions on the max-height property.

The ARIA attributes (aria-expanded, aria-controls, role="region") help improve accessibility for screen reader users.

The button text changes dynamically to reflect the current state (expanded or collapsed).
