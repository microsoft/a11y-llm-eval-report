Usage Instructions ▼

How to use:

  1. Click on the header to expand or collapse the content.
  2. Use keyboard navigation by pressing Enter or Space on focused headers.
  3. The animation provides smooth transitions between states.
  4. Multiple widgets can be used independently on the same page.

Note: This widget is fully accessible and follows ARIA best practices.
