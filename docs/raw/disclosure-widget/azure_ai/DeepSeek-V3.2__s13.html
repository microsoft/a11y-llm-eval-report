Widget 1: Programming Languages

Programming languages are used to create software, websites, and applications. Examples include JavaScript, Python, Java, and C++.

Each language has its own syntax, use cases, and communities. Choosing the right one depends on the project requirements.

Widget 2: Web Development

Web development involves building and maintaining websites. It can be split into frontend (client-side) and backend (server-side).

Frontend technologies include HTML, CSS, and JavaScript. Backend technologies include Node.js, PHP, Ruby on Rails, and databases.

Widget 3: Artificial Intelligence

Artificial Intelligence (AI) refers to machines or software that mimic human intelligence. This includes learning, reasoning, and problem-solving.

Key areas of AI include machine learning, natural language processing, computer vision, and robotics.
