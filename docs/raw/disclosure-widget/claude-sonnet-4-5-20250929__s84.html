Example 2: Frequently Asked Question

What is an expand/collapse widget? ▼

An expand/collapse widget is a user interface component that allows users to show or hide content on demand.

It's commonly used for FAQs, navigation menus, and content that doesn't need to be visible all the time.

This helps keep the interface clean and organized while still providing access to detailed information.
