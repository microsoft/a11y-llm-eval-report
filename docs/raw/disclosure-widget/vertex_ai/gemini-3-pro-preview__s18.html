Expand/Collapse Widgets

This is a simple accordion-style widget. It uses vanilla JavaScript to toggle visibility and ARIA attributes to ensure accessibility for screen readers.

The JavaScript listens for clicks on the button. When clicked, it toggles the is-open class on the content div and updates the aria-expanded attribute on the button.

  • Lightweight
  • Accessible
  • Easy to style
