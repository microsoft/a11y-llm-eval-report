Expand/Collapse Widgets

This is a simple expand/collapse widget. It uses JavaScript to toggle the height of the content container, creating a smooth sliding animation.

The container has a class of details. By default, its max-height is set to 0. When you click the button, JavaScript calculates the scrollHeight of the content and applies it inline, allowing the CSS transition to animate the opening.
