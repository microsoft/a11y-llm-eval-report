Expand/Collapse Widgets

This is a simple expand/collapse widget. It uses vanilla JavaScript to toggle the visibility of content. The animation is handled by transitioning the max-height property.

When you click the header, an event listener triggers.

It calculates the scrollHeight of the content and applies it as an inline style to the container, allowing CSS transitions to animate the height from 0 to the content's natural size.
