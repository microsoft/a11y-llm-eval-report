Expand/Collapse Widgets

This is a simple expand/collapse widget. It uses vanilla JavaScript to toggle the visibility of the content area. The animation is handled by transitioning the max-height property.

When you click the header, a JavaScript function calculates the scrollHeight of the content div and applies it as an inline style. This allows CSS transitions to animate from height 0 to auto.
