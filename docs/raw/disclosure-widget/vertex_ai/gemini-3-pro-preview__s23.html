Expand/Collapse Widgets

This is a simple expand/collapse widget. It uses vanilla JavaScript to toggle the visibility of this content area. The transition is animated using CSS max-height.

The JavaScript listens for clicks on the button. When clicked, it calculates the scrollHeight of the content and applies it as an inline style to the .details container.

This allows for a smooth slide animation even with dynamic content heights.
