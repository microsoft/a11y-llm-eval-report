Modal Dialog Component (example)

This page demonstrates a simple modal dialog. The modal is closed by default. Click the button below (class "trigger") to open it.

Sample modal title

This is a basic modal dialog component. It closes when you click outside, press the Escape key, click the close button, or press the Close action in the footer.

Accessibility: it uses role="dialog" and aria-modal="true". The content is hidden when the modal is closed.
