Modal title

This is an example modal dialog. It is closed by default. You can close it by:

  • Clicking the "Close" button
  • Clicking outside the dialog (the overlay)
  • Pressing the Escape key

Focus is trapped inside the modal while it's open and returned to the trigger when closed.
