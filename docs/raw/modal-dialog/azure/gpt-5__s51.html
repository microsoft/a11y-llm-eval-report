Modal Dialog Component

This example shows a modal dialog that is closed by default. Click the button below to open it.

Welcome to the Modal

This is a simple modal dialog. It uses the native HTML dialog element, supports closing with the Escape key, clicking the close button, or clicking outside the dialog content area.

You can put any content here: forms, text, images, or other interactive elements. Keyboard focus is managed so that interaction outside the dialog is blocked while it is open.
