Modal dialog example

This example shows a modal dialog component that is closed by default. The button below opens the modal. The button that opens the modal has the trigger class.

After opening the modal you can press Tab to navigate, Esc to close, or click outside the dialog to dismiss. Focus is restored to the opener when the modal closes.
