Page with Modal

This is some background content. The modal dialog is closed by default. Press the "Open Modal" button above (it has the class "trigger") to show the dialog.

While the dialog is open, keyboard focus is trapped inside and the background is marked inaccessible to assistive technologies.

Example Modal

This is an example of an accessible modal dialog. It supports:

  • Closing with the Escape key
  • Clicking outside the panel (on the overlay)
  • Focus trapping within the dialog while open
  • Restoring focus to the triggering button when closed
