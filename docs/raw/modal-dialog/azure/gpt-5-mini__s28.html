Modal Dialog Component Example

This page demonstrates an accessible modal dialog. The dialog is closed by default. Use the button below to open it.

Background content remains interactive only when the dialog is closed. When opened, focus is trapped inside the dialog; pressing Escape or clicking outside the panel will close it.

Example Modal

This is an example modal dialog. It demonstrates keyboard accessibility: Tab/Shift+Tab are trapped inside, and pressing Escape closes the dialog. Focus returns to the trigger when the modal closes.
