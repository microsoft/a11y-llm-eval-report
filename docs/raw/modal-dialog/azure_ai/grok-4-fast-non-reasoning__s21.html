Example Modal

This is an example of a modal dialog component. It can be used to display important information, forms, or confirmation dialogs to the user.

Modals are great for focusing user attention on a specific task without navigating away from the current page.

  • Closed by default
  • Opened by clicking the trigger button
  • Closed by clicking the X button or pressing Escape
  • Overlay prevents interaction with background content
