Example Modal

This is an example of a modal dialog component. It includes a header with a title and close button, a body with content, and a footer with action buttons.

Modals are great for displaying important information, forms, or confirmation dialogs without navigating away from the current page.

  • Closed by default
  • Opened by clicking the trigger button
  • Closed by clicking the X button, Cancel button, or clicking outside the modal
  • Responsive design for mobile devices
  • Smooth animations for better UX
