Modal Dialog Example

This page demonstrates a modal dialog component. The modal is hidden by default and can be opened by clicking the button below. The modal includes a header, body content, and a footer with action buttons.

Sample Modal Dialog

This is a sample modal dialog created with HTML, CSS, and JavaScript. It includes a semi-transparent overlay that dims the background content.

You can close this modal by clicking the X button, clicking the overlay outside the modal, or pressing the Escape key on your keyboard.

Modal dialogs are commonly used for notifications, confirmations, forms, or displaying additional information without navigating away from the current page.
