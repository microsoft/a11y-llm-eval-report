Welcome to the Modal!

This is an example of a modal dialog component. It includes:

  • A clickable trigger button with the 'trigger' class
  • An overlay that dims the background
  • A smooth slide-in animation
  • Close functionality via the X button or clicking outside
  • Responsive design for mobile devices

Modals are great for confirmation dialogs, forms, or any content that needs user attention.
