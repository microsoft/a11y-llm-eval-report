Welcome to the Modal

This is an example of a modal dialog component. It appears as a focused overlay on top of the main content, temporarily interrupting the user workflow to present important information or request an action.

You can close this modal by clicking the "Close" button, the "X" icon, or by clicking on the background overlay.
