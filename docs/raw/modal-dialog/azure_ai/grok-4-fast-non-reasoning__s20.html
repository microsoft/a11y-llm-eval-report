Welcome to Our Modal!

This is an example of a modal dialog component. It includes:

  • Smooth animations for opening and closing
  • Accessible keyboard navigation (ESC to close, Tab to focus elements)
  • Responsive design for mobile devices
  • Proper ARIA attributes for screen readers
  • Overlay backdrop that prevents interaction with background content

Modals are great for displaying important information, forms, or confirmation dialogs without navigating away from the current page.
