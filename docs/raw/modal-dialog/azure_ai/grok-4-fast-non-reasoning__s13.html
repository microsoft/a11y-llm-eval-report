Welcome to the Modal

This is an example of a modal dialog component. It includes:

  • A header with title and close button
  • A body with content
  • A footer with action buttons
  • Smooth animations for opening and closing
  • Accessibility features like ARIA attributes

Modals are great for confirming actions, displaying forms, or showing important information that requires user attention.
