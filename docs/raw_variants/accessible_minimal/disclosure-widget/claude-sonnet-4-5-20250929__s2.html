Accessibility Features

This widget implements proper ARIA attributes:

  • aria-expanded: Indicates whether content is expanded or collapsed
  • aria-controls: Associates the button with the content it controls
  • Keyboard accessible with Enter and Space keys
  • Focus indicators for keyboard navigation
