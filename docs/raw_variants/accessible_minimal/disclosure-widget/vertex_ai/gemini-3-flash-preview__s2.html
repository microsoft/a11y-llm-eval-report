Accessible Expand/Collapse Widgets

Web accessibility (A11y) ensures that websites, tools, and technologies are designed and developed so that people with disabilities can use them. This includes people with auditory, cognitive, neurological, physical, speech, and visual disabilities.

This widget uses standard HTML buttons and ARIA attributes:

  • aria-expanded: Informs screen readers if the content is visible.
  • aria-controls: Programmatically links the button to the content container.
  • role="region": Identifies the content area as a significant section of the page.
  • hidden: Properly hides the content from both the screen and assistive technology when collapsed.
