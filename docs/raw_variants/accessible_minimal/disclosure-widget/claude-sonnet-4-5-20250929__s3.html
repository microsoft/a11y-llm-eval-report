Example 2: FAQ Item

An accessible widget is a user interface component that can be used by everyone, including people with disabilities. This includes:

  • Proper keyboard navigation support
  • Screen reader compatibility with ARIA attributes
  • Clear visual focus indicators
  • Sufficient color contrast

Example 3: Technical Details

Specifications

ARIA Attributes Used:

  • aria-expanded: Indicates whether the content is expanded or collapsed
  • aria-controls: Links the button to the content it controls

Keyboard Support:

  • Enter/Space: Toggle expand/collapse state
  • Tab: Navigate between widgets
