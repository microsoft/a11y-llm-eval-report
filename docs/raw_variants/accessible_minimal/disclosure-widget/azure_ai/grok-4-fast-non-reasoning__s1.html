Expand/Collapse Widget Demo

This page demonstrates accessible expand/collapse widgets. Click on the headings to toggle content visibility. Each example is wrapped in a div with the example class, and the controlled content container has the details class.

Example 1: Basic Information + Expand/Collapse

This is the first example of an expand/collapse widget. The content is hidden by default and can be revealed by clicking or pressing Enter/Space on the heading.

The widget uses CSS transitions for smooth animations and includes proper ARIA attributes for screen reader accessibility.

Example 2: Detailed Instructions + Expand/Collapse

Follow these steps to create your own expand/collapse widget:

  1. Create a heading element with appropriate ARIA attributes.
  2. Add a container div with the details class for the collapsible content.
  3. Use JavaScript to toggle the expanded class and update ARIA states.
  4. Ensure keyboard accessibility with tabindex and event handlers.

This example shows how structured content can be organized in collapsible sections.

Example 3: Frequently Asked Questions + Expand/Collapse

What is an expand/collapse widget?

An expand/collapse widget allows users to show or hide content sections interactively, improving page readability and user experience.

Is this accessible?

Yes! This implementation includes ARIA attributes, keyboard navigation, and screen reader support to ensure accessibility compliance.

How does the animation work?

The smooth animation is achieved using CSS transitions on the max-height and padding properties of the details container.
