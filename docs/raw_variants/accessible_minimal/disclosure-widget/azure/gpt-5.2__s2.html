Expand/Collapse Widget Demonstration

Example 1: Native <details> / <summary>

Show more information

This uses the native HTML <details> element, which provides built-in keyboard and screen reader support.

  • Activate the summary to toggle.
  • Works with keyboard (Enter/Space) and pointer.
  • State is exposed to assistive technologies.

Tip: Use Tab to focus the summary, then press Enter or Space.

Example 2: Button controlling a region

The button updates aria-expanded and toggles the region visibility.

Q: What happens when I activate the button?

A: The content region is shown/hidden, and the button’s aria-expanded value is updated.

Keyboard: Tab to the button, then press Enter or Space.
