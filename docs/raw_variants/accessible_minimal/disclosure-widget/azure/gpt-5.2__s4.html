Expand/Collapse Widget Demo

Example 1: Basic details panel

This is the controlled content. It is hidden by default and becomes visible when the button is activated.

  • Keyboard accessible (Tab to focus, Enter/Space to toggle).
  • Screen-reader friendly via ARIA relationships.
  • Uses the details class as requested.

Tip: Press Tab to focus the button, then press Enter or Space.
