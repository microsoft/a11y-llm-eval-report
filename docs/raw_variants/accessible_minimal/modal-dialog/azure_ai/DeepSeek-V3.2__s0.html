This page demonstrates a modal dialog component. The modal is closed by default. Click the button below to open it. The modal can be closed using the close button, the overlay, or the Escape key.

Sample Modal

This is an example of a modal dialog component. It appears on top of the page content and requires user interaction to dismiss. You can close this modal by clicking the X button, clicking the background overlay, pressing the Escape key, or using the confirmation button below.

Modal dialogs are useful for focusing user attention on important information or actions without navigating away from the current page.
