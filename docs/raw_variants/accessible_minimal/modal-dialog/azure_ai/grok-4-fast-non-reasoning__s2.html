Sample Modal Dialog

This is an example of an accessible modal dialog component. The modal is closed by default and can be opened using the trigger button above.

Key accessibility features include:

  • Proper ARIA roles and labels
  • Keyboard navigation support
  • Focus management (traps focus within the modal when open)
  • Screen reader announcements
  • High contrast focus indicators
  • Responsive design for mobile devices

You can close this modal by clicking the close button (×), pressing the Escape key, or clicking outside the modal.
