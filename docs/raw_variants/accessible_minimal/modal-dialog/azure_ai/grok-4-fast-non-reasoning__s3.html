Sample Modal Dialog

This is an example of an accessible modal dialog. It includes proper ARIA attributes, keyboard navigation, and focus management.

Features:

  • Opened by clicking the trigger button
  • Closed by clicking the close button or pressing Escape
  • Keyboard accessible (Tab to navigate, Escape to close)
  • Screen reader friendly with proper ARIA roles
  • Responsive design for mobile devices
  • Focus is trapped within the modal when open
