Modal dialog component example

Try the dialog

This example shows a modal dialog that opens on demand and returns focus to the trigger when closed.

What happens in this dialog

  • Focus moves into the dialog when it opens.
  • Pressing Escape or the close button will close it.
  • Focus returns to the “Open dialog” button after closing.
