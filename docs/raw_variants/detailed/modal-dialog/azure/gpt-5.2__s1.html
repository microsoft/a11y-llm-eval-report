Modal dialog component

Activate the “Open dialog” button to open a modal dialog. Press Escape or use the Close button to dismiss it.

What this example includes

  • Modal behavior using the native <dialog> element.
  • Focus moves into the dialog on open and returns to the trigger on close.
  • Keyboard support: Tab/Shift+Tab stays within the dialog; Escape closes.
