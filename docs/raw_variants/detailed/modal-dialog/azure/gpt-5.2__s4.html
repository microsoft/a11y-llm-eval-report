Modal dialog component

Example

Activate the button to open a modal dialog. While open, focus stays within the dialog. Press Escape or use the Close button to dismiss it.

Notes

  • The dialog is closed by default.
  • The open button includes the required trigger class.
  • The dialog uses the native <dialog> element for built-in semantics and focus handling.
