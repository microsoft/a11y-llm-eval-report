Modal dialog component

This example uses the native <dialog> element. The dialog is closed by default. Activate the button to open it. Press Escape or use the Close button to dismiss it.

Notes

The dialog traps focus while open (native behavior with showModal()), restores focus to the trigger on close, and prevents background scrolling while open.
