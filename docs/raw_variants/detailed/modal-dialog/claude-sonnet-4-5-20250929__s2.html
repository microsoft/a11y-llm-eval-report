Modal Dialog Example

This page demonstrates an accessible modal dialog component. The modal is closed by default and can be opened by clicking the button below.

The modal implements proper focus management, keyboard navigation, and ARIA attributes to ensure accessibility for all users.

Example Modal Dialog

This is an example modal dialog. It demonstrates proper accessibility features including:

  • Focus is trapped within the modal while open
  • Pressing Escape closes the modal
  • Focus returns to the trigger button when closed
  • Background content is inert while modal is open
  • Proper ARIA attributes for screen readers

You can interact with the buttons below or press Escape to close this dialog.
