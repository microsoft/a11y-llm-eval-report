Modal Dialog Example

This page demonstrates an accessible modal dialog component.

About This Modal

The modal dialog can be opened by clicking the button above. Once open:

  • Focus is automatically moved to the modal
  • You can close it by clicking the close button or the "Cancel" button
  • Pressing the Escape key will also close the dialog
  • Focus is returned to the trigger button when closed
  • All content outside the modal is inert while it's open
