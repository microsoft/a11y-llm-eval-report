Accessibility Features

  • Uses the <dialog> element with correct ARIA attributes.
  • Focus is managed: sent to the dialog on open, returned to trigger on close.
  • Keyboard trap within the modal (Esc closes, Tab cycles).
  • Screen reader announcements via aria-labelledby and aria-describedby.
  • High contrast and forced colors mode support.
  • Reflows correctly for narrow viewports (320px).
