Example Modal Dialog

This is an example modal dialog. It demonstrates proper accessibility features including:

  • Focus management (focus moves to the dialog when opened)
  • Keyboard support (Escape key closes the dialog)
  • Focus trap (Tab and Shift+Tab cycle through dialog elements)
  • Screen reader support (proper ARIA attributes)

When the dialog closes, focus returns to the button that opened it.
