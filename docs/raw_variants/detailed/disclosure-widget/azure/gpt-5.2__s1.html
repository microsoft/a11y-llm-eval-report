Expand/collapse widget examples

These examples show two common patterns for revealing and hiding content. Both are keyboard operable and keep labels visible.

Example 1: Native <details> and <summary>

What information is included?

This section contains additional context that is hidden until you expand it. The native <details> element provides built-in semantics and keyboard support in modern browsers.

  • Keyboard: Tab to the summary, then press Enter or Space to toggle.
  • Screen readers: The expanded/collapsed state is announced.
  • Reflow: Content wraps and stays readable at narrow widths.
