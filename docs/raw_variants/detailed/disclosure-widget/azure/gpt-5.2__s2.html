Expand/collapse widget examples

These examples show two common patterns. Use Tab to move focus and Enter or Space to toggle.

Example 1: Native <details> / <summary>

What information is included?

This content is inside a native <details> element. It provides built-in keyboard and screen reader support.

  • It can be expanded and collapsed.
  • The summary text stays visible as the control label.
  • Content reflows on small screens without horizontal scrolling.

Tip: The summary element is the interactive control.

Example 2: Button toggling a controlled region

Collapsed.

This region is controlled by the button using aria-controls and aria-expanded. When collapsed, the region is removed from the tab order using hidden.

You can place any content here, such as additional instructions, settings, or a short explanation.

Keyboard: Activate the button with Enter or Space. Pressing Escape while focus is inside the expanded panel will collapse it.

Note: Manual testing with a keyboard and assistive technology is recommended for your specific use case.
