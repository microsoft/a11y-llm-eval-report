Example 1: Native <details> (recommended when it fits)

What information is included in this section?

This content is revealed when the summary is expanded. The native <details> element provides built-in keyboard and screen reader support.

  • Keyboard: use Enter or Space on the summary to toggle.
  • Reflow: the layout adapts to narrow screens without horizontal scrolling.
  • Plain language: short sentences and clear headings.

Notes

If the content is hidden, it is not focusable. When expanded, focus stays where you left it; you can optionally move focus into the revealed content if that better matches your use case.

If you notice unexpected behavior with assistive technology, test with multiple browsers and screen readers.
