Expand/Collapse Widget Demo

Each section below can be expanded or collapsed by activating its button. Use Enter or Space to toggle.

Examples

An expand/collapse widget (also known as a disclosure widget) allows users to show or hide a section of content. It consists of a button that toggles the visibility of an associated content region.

This pattern is useful for reducing visual clutter while keeping content accessible on demand.

When building an expand/collapse widget, keep the following in mind:

  • Use a <button> element as the toggle control so it is keyboard operable by default.
  • Set aria-expanded on the button to communicate the current state to assistive technology.
  • Use aria-controls to associate the button with the content it controls.
  • Use the hidden attribute to hide collapsed content from all users, including assistive technology.
  • Ensure the focus indicator is clearly visible and meets contrast requirements.

The following keyboard interactions are supported:

  • Enter or Space — Toggles the expanded/collapsed state of the associated content.
  • Tab — Moves focus to the next focusable element in the page.
  • Shift + Tab — Moves focus to the previous focusable element in the page.

Because native <button> elements are used, no additional keyboard handling is required.
