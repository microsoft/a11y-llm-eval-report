Expand/Collapse Widget Examples

The following examples demonstrate accessible patterns for toggling the visibility of content.

1. Custom Disclosure Pattern

Using a standard <button> with aria-expanded and aria-controls.

Web accessibility means that websites, tools, and technologies are designed and developed so that people with disabilities can use them.

This includes people with auditory, cognitive, neurological, physical, speech, and visual disabilities.

2. Native HTML Details Pattern

Using the native <details> and <summary> elements.

How does the native details element work?

The <details> element creates a disclosure widget in which information is visible only when the widget is toggled into an "open" state.

The <summary> element provides the visible heading for the widget.
