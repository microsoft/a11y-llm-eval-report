Expand/Collapse Widget Demo

Each section below can be expanded or collapsed by activating its heading button. Use Enter or Space to toggle.

Examples

An expand/collapse widget (also known as a disclosure widget) allows users to show or hide a section of content. It consists of a button that toggles the visibility of an associated content region.

This pattern is useful for reducing visual clutter while keeping content accessible on demand.

The toggle button is a standard <button> element, so it is natively keyboard accessible:

  • Tab moves focus to the button.
  • Enter or Space activates the button, toggling the content visibility.

The aria-expanded attribute on the button communicates the current state to assistive technologies.

Consider using an expand/collapse widget when:

  • You have supplementary content that not all users need to see immediately.
  • A page has many sections and you want to reduce scrolling.
  • You are building a frequently asked questions (FAQ) section.
  • You want to progressively disclose information to reduce cognitive load.

Avoid hiding critical content that users need to complete a primary task.
