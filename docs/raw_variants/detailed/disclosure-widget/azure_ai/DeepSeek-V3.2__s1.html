Core Requirements

  • All interactive elements must be keyboard operable.
  • Provide sufficient color contrast (4.5:1 for normal text).
  • Use semantic HTML elements and ARIA only when necessary.
  • Ensure all form controls have associated visible labels.

Additional Guidelines

Content should reflow to a 320px viewport without horizontal scrolling for text. Support forced colors mode. Manage focus for dynamic content updates.
