FAQ Item

An accessible expand/collapse widget is a user interface component that allows users to show or hide content on demand. It must be keyboard accessible, provide proper ARIA attributes, and work with screen readers.

Key features include:

  • Keyboard support (Enter and Space keys)
  • Proper aria-expanded state
  • Clear visual indicators
  • Logical focus management
