Expand/Collapse Widget Demo

FAQ Item 1

This is an accessible expand/collapse widget that allows users to show or hide additional content. It uses semantic HTML and ARIA attributes for full keyboard and screen reader support.

FAQ Item 2

The widget uses a button with ARIA attributes to control the visibility of content in a container with the 'details' class. JavaScript toggles the aria-expanded and aria-hidden states, and CSS handles the smooth animation.

  • Click or press Enter/Space to toggle
  • Visual indicator rotates when expanded
  • Smooth height transition for accessibility
