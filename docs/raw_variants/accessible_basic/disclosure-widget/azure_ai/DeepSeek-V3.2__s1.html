Expand/Collapse Widget Demo

Project Overview

This project aims to create an accessible and user-friendly interface component library. The expand/collapse widget is a fundamental pattern for managing content visibility without overwhelming the user.

Key features include full keyboard navigation support, clear visual focus indicators, and appropriate ARIA attributes for screen reader users. The animation is smooth and uses CSS transitions for performance.

Technical Specifications

HTML: Semantic structure with button and div elements. ARIA attributes dynamically updated via JavaScript.

CSS: Responsive design with focus states, smooth transitions for height and opacity, and a visually hidden class for screen reader text.

JavaScript: Lightweight script to toggle aria-expanded, aria-hidden, and manage CSS classes for the animation. Event listeners for both click and keyboard (Enter, Space).

Accessibility: WCAG 2.2 AA compliant. Supports keyboard interaction, screen readers, and has sufficient color contrast.

Usage Instructions

1. Click the button or press Enter/Space while focused to expand or collapse the content.

2. The button text and icon change contextually to indicate the action (Show/Hide).

3. The content area expands smoothly. Screen readers will announce the expanded state.

4. Ensure each widget has a unique id for the aria-controls and aria-labelledby relationships.
