Expand/Collapse Widget Demo

FAQ Item 1

This is an accessible expand/collapse widget that demonstrates how to create interactive content sections using semantic HTML, ARIA attributes, and smooth transitions. It supports keyboard navigation and screen readers.

FAQ Item 2

The widget uses a button as the trigger with proper ARIA attributes (aria-expanded and aria-controls) to manage state and relationships. The content is hidden/shown using CSS transitions on max-height, ensuring smooth animations while maintaining accessibility.

  • Click or press Enter/Space to toggle
  • Focus is indicated with an outline
  • Screen reader announces state changes
