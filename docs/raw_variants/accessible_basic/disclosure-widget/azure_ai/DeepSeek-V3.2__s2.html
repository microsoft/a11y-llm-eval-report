This project builds a responsive web application for managing tasks. It includes features like drag-and-drop organization, real-time collaboration, and deadline tracking. The frontend uses modern frameworks, while the backend leverages a scalable cloud infrastructure.

Key technologies: HTML, CSS, JavaScript, Node.js, and PostgreSQL.
