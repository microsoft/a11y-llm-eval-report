Accessible Disclosure Widgets

Native HTML5 Implementation

What is the benefit of using the native <details> element?

The native element handles keyboard interaction (Space/Enter) and state management (open/closed) automatically without requiring JavaScript. It is highly accessible by default.

Custom ARIA Implementation

To make a custom widget accessible, you must:

  • Use a <button> for the trigger.
  • Use aria-expanded to communicate state.
  • Use aria-controls to link the button to the content.
  • Ensure the content is hidden from screen readers and keyboard focus when collapsed using the hidden attribute or display: none.
