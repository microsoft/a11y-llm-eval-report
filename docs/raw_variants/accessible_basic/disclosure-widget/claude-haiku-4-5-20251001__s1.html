What is semantic HTML?

Semantic HTML is the use of HTML markup to reinforce the meaning of the content rather than merely to define its appearance. It provides meaning to web pages that aids both the web browser and the developer.

Examples of semantic elements include <header>, <nav>, <main>, <article>, and <footer>.

What is accessibility?

Web accessibility means that websites, tools, and technologies are designed and developed so that people with disabilities can use them. More specifically, people can perceive, understand, navigate, and interact with the Web.

What is WCAG?

The Web Content Accessibility Guidelines (WCAG) are part of a series of web accessibility guidelines published by the World Wide Web Consortium (W3C). They lay out recommendations for making web content more accessible to people with disabilities.

What is keyboard navigation?

Keyboard navigation allows users to interact with web content using only their keyboard. This is essential for users who cannot use a mouse, including those with motor disabilities or those using assistive technologies.
