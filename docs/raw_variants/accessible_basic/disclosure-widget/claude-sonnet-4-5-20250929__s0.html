Example 2: FAQ Item

An accessible expand/collapse widget is an interactive component that allows users to show or hide content on demand. It must be:

  • Keyboard accessible (operable with Enter and Space keys)
  • Properly labeled with ARIA attributes
  • Announcing state changes to screen readers
  • Visually indicating expanded/collapsed state
