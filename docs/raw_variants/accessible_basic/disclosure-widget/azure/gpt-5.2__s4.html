Expand/Collapse Widget

Accessible examples using native <details> and <summary>.

Example 1: Single expandable panel

What is this widget?

This is a native expand/collapse control. It supports keyboard interaction by default: use Tab to focus the summary, then press Enter or Space to toggle.

Because it’s native HTML, it works well with assistive technologies and requires no JavaScript.

Tip: Try toggling with the keyboard and observe the focus outline.

Example 2: FAQ list (multiple panels)

Can more than one panel be open?

Yes. Each <details> element is independent, so multiple panels can be open at once.

Can I include links and lists inside?

Yes. Content can include typical elements like lists and links. For example, visit the HTML specification.

  • Lists remain fully accessible.
  • Links are keyboard focusable.
  • Text reflows for zoom and small screens.
How do I start a panel open?

Add the open attribute to the <details> element. (This example is closed by default.)
