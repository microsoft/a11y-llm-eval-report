Expand/Collapse Widget Demonstration

Two accessible patterns: native <details> and a button controlling a content region.

Example 1: Native details/summary

What information is inside this panel?

This content is controlled by the native <details> element.

  • Keyboard: Tab to the summary, then press Enter or Space to toggle.
  • Screen readers announce expanded/collapsed state automatically.
  • No JavaScript required.

Example 2: Button controlling a region

Tip: Press Enter or Space to toggle.

This panel is controlled by a button using aria-expanded and aria-controls.

It is hidden using the hidden attribute when collapsed, and shown when expanded. Focus remains on the toggle button for predictable keyboard use.

Learn more about accessible disclosure patterns: WAI-ARIA Authoring Practices: Disclosure Pattern .

All examples are designed for keyboard and screen reader compatibility and aim to meet WCAG 2.2 Level AA.
