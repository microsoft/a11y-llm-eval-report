Example 2: FAQ Section

An accessible expand/collapse widget is an interactive component that allows users to show or hide content. It must meet several requirements:

  • Uses semantic HTML elements appropriately
  • Includes proper ARIA attributes like aria-expanded and aria-controls
  • Supports full keyboard navigation
  • Provides clear visual focus indicators
  • Announces state changes to screen readers
