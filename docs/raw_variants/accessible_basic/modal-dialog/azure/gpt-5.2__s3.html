Modal dialog example

This example uses the native <dialog> element. The dialog is closed by default. Use the “Open dialog” button to show it. Press Esc to close.

Keyboard: Tab moves focus; Shift+Tab moves backward. When the dialog opens, focus moves inside it and is trapped until closed.

Example modal dialog

This is a modal dialog. It traps focus while open and returns focus to the trigger button when closed.
