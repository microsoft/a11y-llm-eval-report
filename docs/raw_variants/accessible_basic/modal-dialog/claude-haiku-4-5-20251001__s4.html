Modal Dialog

This is an example of an accessible modal dialog component. It follows WCAG 2.2 Level AA guidelines and includes proper keyboard support.

You can close this modal by:

  • Clicking the close button
  • Clicking the Cancel button
  • Pressing the Escape key
  • Clicking outside the modal

Focus is automatically trapped within the modal while it's open, and returns to the trigger button when closed.
