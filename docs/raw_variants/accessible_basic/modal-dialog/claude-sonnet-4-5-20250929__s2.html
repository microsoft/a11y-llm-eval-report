Modal Dialog Example

This is an accessible modal dialog component. It follows WCAG 2.2 Level AA guidelines and includes:

  • Keyboard navigation support (Tab, Shift+Tab, Escape)
  • Focus management and trapping
  • Proper ARIA attributes
  • Screen reader announcements
  • Visible focus indicators

You can close this modal by clicking the close button, clicking outside the dialog, or pressing the Escape key.
