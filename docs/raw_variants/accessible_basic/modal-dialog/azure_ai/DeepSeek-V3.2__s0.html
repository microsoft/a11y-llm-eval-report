This page demonstrates an accessible modal dialog that is closed by default. The button below will open the modal when activated.

Modal Dialog Title

This is an example of an accessible modal dialog. It traps focus inside when open and returns focus to the triggering button when closed.

You can close this modal by pressing the close button, pressing the Escape key, or clicking outside the modal.
