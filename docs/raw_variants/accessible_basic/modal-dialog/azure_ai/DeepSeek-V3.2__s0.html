Accessible Modal Dialog Example

This page demonstrates a modal dialog that meets WCAG 2.2 Level AA requirements. The button below will open the modal.

Modal Dialog Title

This is a modal dialog window. It interrupts the main content to present important information or request an action.

This is the main content of the modal dialog. It can contain text, form controls, or other interactive elements.

For accessibility, the dialog traps keyboard focus, is announced by screen readers, and can be closed with the Escape key.

  • Focus is moved into the dialog when opened.
  • Focus is returned to the trigger button when closed.
  • Clicking the overlay or pressing Escape closes the dialog.
