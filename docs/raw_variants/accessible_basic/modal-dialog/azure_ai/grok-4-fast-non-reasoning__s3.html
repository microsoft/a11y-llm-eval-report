Modal Dialog Example

This is an example of an accessible modal dialog. It follows WCAG 2.2 Level AA guidelines, including proper ARIA attributes, keyboard navigation, and focus management.

  • Focus is trapped within the modal when open.
  • Escape key closes the modal.
  • Clicking the overlay or close button dismisses it.
  • Screen reader announcements are supported.
